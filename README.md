Light Controller
================

使用 Arduino 来控制一个 LED 灯的亮度。

使用 Processing 编写的可视化控制条，用户可以通过拉拽控制条来调整亮度的数值，通过串口将亮度输送给 Arduino，由 Arduino 实现 LED 灯亮度的调整。

已经测试成功。
